# Part 2: NoSQL Analysis and MongoDB Implementation

This section evaluates the suitability of MongoDB for handling flexible and diverse product data.

## Objectives

- Analyze limitations of relational databases for dynamic product attributes
- Justify the use of MongoDB for product catalogs
- Implement common MongoDB operations and aggregations

## What Was Implemented

- Written analysis comparing RDBMS and MongoDB
- MongoDB collection storing products with nested specifications and reviews
- Queries for filtering, aggregation, and updates
- Average rating and price analytics using aggregation pipelines

## Files

- `nosql_analysis.md` – Theory and justification report
- `mongodb_operations.js` – MongoDB operations script
- `products_catalog.json` – Sample product data

## Tools Used

- MongoDB 8.2.3
- MongoDB Compass
- mongosh
